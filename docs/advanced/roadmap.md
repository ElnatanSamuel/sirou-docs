# Roadmap

Our goal is to make Sirou the standard for universal routing in the TypeScript ecosystem. Here is what we are working on:

## 🟢 Q1 2026: The Stability Phase (Current)

- [x] **Universal Core**: Stable Radix Trie matcher.
- [x] **React & Next.js Adapters**: Production-ready hooks and components.
- [x] **Data Loaders**: Fully integrated data lifecycle.
- [x] **CLI Portal**: Interactive route reference generator.

## 🟡 Q2 2026: Ecosystem Expansion

- [ ] **Zod Integration**: Native support for Zod schemas in `params` for runtime validation.
- [ ] **Internationalization (i18n)**: Automatic path prefixing (e.g., `/en/home`, `/fr/accueil`) based on user locale.
- [ ] **React Native Navigation V7 Support**: Aligning with the latest mobile navigation standards.

## 🟠 Q3 2026: Performance & Scale

- [ ] **Edge Routing**: Optimized adapters for Cloudflare Workers and Vercel Edge.
- [ ] **Path Compression**: Specialized algorithms for super-large schemas (1000+ routes).
- [ ] **Visual Schema Designer**: A GUI for defining and managing routes visually.

## 🔴 Future Vision

- [ ] **AI-Assisted Navigation**: Predicting which routes a user will visit based on behavior and pre-loading data automatically.
- [ ] **Universal SDK Generator**: Automatically generate API clients derived from your route loaders.
